## Introduction

The Microplate AI System is an advanced platform for hemagglutination inhibition analysis. This manual provides step-by-step instructions for using the system.

## Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Valid user account

### Accessing the System
1. Navigate to the application URL
2. You will be redirected to the login page if not authenticated

## Authentication

### Login
1. Enter your email address
2. Enter your password
3. Click "Login" button
4. You will be redirected to the dashboard upon successful login

### Logout
1. Click on your profile icon in the top-right corner
2. Select "Logout" from the dropdown menu

### Password Reset
1. Click "Forgot Password" on the login page
2. Enter your email address
3. Check your email for reset instructions
4. Follow the link to reset your password

## Image Capture

### Capture New Image
1. Navigate to the Capture page from the dashboard
2. Enter the following information:
   - Sample Number (required)
   - Submission Number (optional)
   - Description (optional)
3. Choose one of the following options:
   - **Live Capture**: Click "Capture" to capture from camera
   - **Upload Image**: Click "Upload" to select an image file
4. Wait for processing to complete
5. View prediction results

### Viewing Captured Images
- Captured images are displayed in the results section
- Annotated images show detected wells and predictions

## Viewing Results

### Sample Results
1. Navigate to the Results page
2. Use the search bar to find a sample by number
3. Click on a sample to view detailed results
4. View confidence scores and predictions for each well

### Exporting Results
1. Navigate to the Results page
2. Select a sample
3. Click "Export" button
4. Choose export format (CSV or JSON)
5. Download the file

## Settings

### Language Selection
1. Click on your profile icon
2. Select language from the dropdown:
   - English
   - ไทย (Thai)

### Theme Selection
1. Navigate to Settings page
2. Select your preferred theme:
   - Light
   - Dark
   - System (follows OS preference)

### Profile Settings
1. Navigate to Profile page
2. Update your information:
   - Username
   - Email
   - Password
3. Click "Save" to apply changes

## Troubleshooting

### Login Issues
- Verify your email and password are correct
- Check your internet connection
- Clear browser cache and cookies
- Contact support if problems persist

### Image Upload Issues
- Verify image format is supported (JPEG, PNG, WebP, TIFF)
- Check file size is within limits (50MB)
- Ensure you have proper permissions

### Performance Issues
- Clear browser cache
- Check internet connection speed
- Try a different browser
- Contact support if issues continue
